# asteroids

A [boot.dev](boot.dev) game project

## Usage

1. Use Python >=3.12 (e.g. with `uv`)

   ```sh
   uv venv
   uv sync
   ```

2. Run

   ```sh
   uv run main.py
   ```

   or

   ```sh
   source .venv/bin/activate
   python main.py
   ```

## Controls

- `WASD`/`arrow keys`: move
- `Space`: shoot
- `q`: exit game
